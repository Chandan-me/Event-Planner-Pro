# 🎉 Event-Planner-Pro

A modern, AI-powered event management web application built using **Next.js**, **Tailwind CSS**, and **Framer Motion**.  
This platform allows users to explore, book, and manage events seamlessly with a premium UI experience.

## 🛠️ Installation & Setup

### 1️⃣ Clone the repository
```bash
git clone https://github.com/your-username/Event-Planner-Pro.git
cd Event-Planner-Pro
```

### 2️⃣ Install dependencies
```bash
npm install
```

### 3️⃣ Run the development server
```bash
npm run dev
```
